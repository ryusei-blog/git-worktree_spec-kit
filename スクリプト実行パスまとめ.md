# スクリプト実行パスまとめ

## 新規プロジェクト用（`new-project-type.zsh`）

```zsh
cd /Users/archi/Documents/GitHub
zsh /Users/archi/Documents/GitHub/git-worktree_spec-kit/new-project-type.zsh
```

### スクリプト後に対話シェルを開きたくない場合

```zsh
NEW_PROJECT_TYPE_NO_SHELL=1 zsh /path/to/new-project-type.zsh
```

## 既存プロジェクト用（`existing-project.zsh`）

```zsh
zsh /Users/archi/Documents/GitHub/git-worktree_spec-kit/existing-project.zsh
```

- **対象となる既存リポジトリに事前に cd した上で実行する。**

### 対話シェルを開きたくない場合

```zsh
NEW_PROJECT_TYPE_NO_SHELL=1 zsh /path/to/existing-project.zsh
```
